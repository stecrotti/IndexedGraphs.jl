IndexedGraphs.FactorGraph — Type
FactorGraph{T<:Integer} <: AbstractIndexedGraph{T}

A type representing a sparse factor graph.

FIELDS

  • A – adjacency matrix filled with NullNumbers. Rows are factors, columns are variables
  • X – square matrix for efficient access by row. X[j,i] points to the index of element A[i,j] in A.nzval.
source
IndexedGraphs.FactorGraphEdge — Type

FactorGraphEdge{T<:Integer} <: AbstractIndexedEdge{T}

Edge type for FactorGraphs.

Safe constructors

FactorGraphEdge(v::Variable, f::Factor, idx::Integer)

Example

Note

Specifying the variable first is also valid: FactorGraphEdge(v::Variable, f::Factor, idx::Integer)
